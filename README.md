## Описание проекта
Web приложение, аналог простейшего чата в комнате.
Пользователи заходят в чат и оставляют в нем свои сообщения.

## Используемые технологии
+ **Java 14**
+ **Spring (Boot, Data, MVC, Security (JWT))**
+ **Swagger 2**
+ **PostgreSQL**, **Hibernate**
+ **Maven**
